Word域应用

发表于 2020-10-12 更新于 2020-10-13 分类于 Tech ， Software

简介

简单来说就是Word文档中的信息代码，类似于Excel中的函数功能，善用域可以省去许多重复性操作。

特征字符：{}

关系式：{ Name [参数] [\Switches] }

​ \Switches：域开关

打开方式：

  1. 插入 — 文档部件 — 域
  2. Ctrl + F9

注：

  • 大括号不可手动输入，必须使用Ctrl + F9添加
  • 域大括号内侧各保留一个空格
  • 域名与其属性或开关之间保留一个空格

快捷键

快捷键 功能
Ctrl + F9 插入域
F9 更新选定的域
Shift + F9 在所选的域与结果之间切换
Alt + F9 在所有的域与结果之间切换
F11 定位下一个域
Shift + F11 定位上一个域
Alt + Shift + D 插入日期域
Alt + Shift + P 插入页字段
Alt + Shift + T 插入时间域
